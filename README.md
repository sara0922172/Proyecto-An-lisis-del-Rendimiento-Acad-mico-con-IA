# Proyecto-An-lisis-del-Rendimiento-Acad-mico-con-IA
Analizar qué factores influyen en el rendimiento académico de los estudiantes utilizando técnicas de minería de datos. Se busca identificar la relación entre variables como horas de estudio, asistencia, horas de sueño, uso de redes sociales y promedio anterior con respecto a la nota final.
