# Proyecto-An-lisis-del-Rendimiento-Acad-mico-con-IA
Analizar qué factores influyen en el rendimiento académico de los estudiantes utilizando técnicas de minería de datos. Se busca identificar la relación entre variables como horas de estudio, asistencia, horas de sueño, uso de redes sociales y promedio anterior con respecto a la nota final.
# Proyecto: Minería de Datos con IA

## Objetivo
Analizar qué factores influyen en el rendimiento académico utilizando técnicas de minería de datos.

---

## Dataset
El dataset contiene 50 registros con las siguientes variables:

- horas_estudio
- asistencia
- horas_sueno
- uso_redes
- promedio_anterior
- nota_final

---

## Limpieza de Datos
Se realizó en Excel:

- Verificación de valores nulos.
- Eliminación de duplicados.
- Validación de formatos.
- Revisión de rangos.

---

## Técnica Aplicada
Se utilizó Regresión Lineal Múltiple en Excel (Analysis ToolPak).

Variable dependiente:
- nota_final

Variables independientes:
- horas_estudio
- asistencia
- horas_sueno
- uso_redes
- promedio_anterior

---

## Resultados
- Relación positiva entre horas de estudio y nota final.
- Relación positiva entre asistencia y rendimiento.
- Relación negativa entre uso de redes y nota.
- Promedio anterior es un factor importante.

---

## Conclusiones
El rendimiento académico depende principalmente del tiempo de estudio, asistencia y desempeño previo.

---

## Limitaciones
- Dataset simulado.
- Número reducido de registros.
- No incluye factores externos.

---

## Recomendaciones
- Aplicar el modelo a datos reales.
- Usar más variables.
- Probar otras técnicas de minería de datos.
